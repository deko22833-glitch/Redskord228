# Мессенджер

Полнофункциональный мессенджер с регистрацией, системой друзей и видеозвонками, работающий полностью на клиентской стороне.

## Возможности

- ✅ Регистрация новых пользователей (без почты)
- ✅ Вход в систему
- ✅ Система друзей (добавление и удаление)
- ✅ Приватные чаты с друзьями
- ✅ Видеозвонки через WebRTC
- ✅ Голосовые звонки
- ✅ Управление видео/аудио во время звонка
- ✅ Современный и красивый интерфейс
- ✅ Адаптивный дизайн

## Установка и запуск

1. Клонируйте репозиторий или скачайте файлы
2. Откройте `index.html` в браузере
3. Зарегистрируйтесь или войдите в систему
4. Начните общаться!

## Развертывание на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы в репозиторий
3. Перейдите в Settings → Pages
4. Выберите ветку `main` и папку `/ (root)`
5. Сохраните изменения
6. Ваш сайт будет доступен по адресу: `https://ваш-username.github.io/имя-репозитория/`

## Технологии

- HTML5
- CSS3 (с градиентами и анимациями)
- Vanilla JavaScript
- LocalStorage для хранения данных

## Примечание

Данные хранятся в LocalStorage браузера, поэтому они доступны только на том устройстве, где вы зарегистрировались. Для работы на нескольких устройствах потребуется серверная часть.

**Звонки:** Видео и голосовые звонки работают через WebRTC. Для полноценной работы между разными устройствами рекомендуется использовать signaling сервер. В текущей реализации звонки работают через localStorage как механизм signaling, что подходит для демонстрации и локального использования.
